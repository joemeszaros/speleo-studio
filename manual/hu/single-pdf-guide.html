Egyetlen PDF fájl létrehozása

Ez az útmutató segít egyetlen PDF fájlt létrehozni, amely tartalmazza a teljes Speleo Studio felhasználói kézikönyvet. Minden módszer automatikusan összefűzi az összes oldalt egy PDF-be.

1. wkhtmltopdf - Ajánlott megoldás Közepes

Automatikus összefűzés minden oldallal

A wkhtmltopdf eszköz képes automatikusan összefűzni az összes HTML oldalt egy PDF-be, beleértve a tartalomjegyzéket és navigációs linkeket is.

  1. Töltse le és telepítse a wkhtmltopdf-et: https://wkhtmltopdf.org/downloads.html
  2. Nyissa meg a terminált/parancssort
  3. Navigáljon a kézikönyv mappájába: cd manual
  4. Futtassa a következő parancsot:
wkhtmltopdf --enable-local-file-access --page-size A4 --margin-top 20mm --margin-bottom 20mm --margin-left 15mm --margin-right 15mm --header-html header.html --footer-html footer.html index.html 01-bevezetes.html 02-projekt-kezeles.html 03-adatok-importalasa.html 04-3d-vizualizacio.html 05-felmereek-szerkesztese.html 06-attributumok.html 07-eszkozok.html 08-exportalas.html 09-beallitasok.html 10-tippek-trukkok.html speleo-studio-teljes-kezikonyv.pdf
Megjegyzés: Ez a parancs létrehoz egy teljes PDF-et az összes oldallal, megfelelő oldalszámozással és formázással.

Előnyök:

  • Egyetlen PDF fájl
  • Kiváló minőség
  • Teljes CSS támogatás
  • Automatikus tartalomjegyzék
  • Oldalszámozás

Hátrányok:

  • Telepítés szükség
  • Parancssori használat
  • Technikai ismeretek kellenek

2. Puppeteer Script - Fejlett megoldás Haladó

Node.js alapú automatikus összefűzés

Létrehozhat egy Node.js scriptet, amely automatikusan összefűzi az összes oldalt és létrehoz egy teljes PDF-et.

  1. Telepítse a Node.js-t
  2. Hozzon létre egy új mappát a projektben
  3. Futtassa: npm init -y
  4. Telepítse a Puppeteer-t: npm install puppeteer
  5. Hozzon létre egy generate-pdf.js fájlt
const puppeteer = require('puppeteer'); const fs = require('fs'); const pages = [ 'manual/index.html', 'manual/01-bevezetes.html', 'manual/02-projekt-kezeles.html', 'manual/03-adatok-importalasa.html', 'manual/04-3d-vizualizacio.html', 'manual/05-felmereek-szerkesztese.html', 'manual/06-attributumok.html', 'manual/07-eszkozok.html', 'manual/08-exportalas.html', 'manual/09-beallitasok.html', 'manual/10-tippek-trukkok.html' ]; (async () => { const browser = await puppeteer.launch(); const page = await browser.newPage(); // Összefűzött PDF létrehozása await page.goto(`file://${process.cwd()}/manual/index.html`, { waitUntil: 'networkidle0' }); await page.pdf({ path: 'speleo-studio-teljes-kezikonyv.pdf', format: 'A4', margin: { top: '20mm', bottom: '20mm', left: '15mm', right: '15mm' }, printBackground: true, displayHeaderFooter: true, headerTemplate: '<div style="font-size: 10px; text-align: center; width: 100%;">Speleo Studio Felhasználói Kézikönyv</div>', footerTemplate: '<div style="font-size: 10px; text-align: center; width: 100%;">Oldal <span class="pageNumber"></span> / <span class="totalPages"></span></div>' }); await browser.close(); console.log('PDF sikeresen létrehozva!'); })();
Használat: Futtassa a scriptet: node generate-pdf.js

3. Online HTML összefűző szolgáltatások Könnyű

Webes eszközök több oldal összefűzésére

Néhány online szolgáltatás támogatja több HTML oldal összefűzését egy PDF-be.

  1. Használja a PDF24 online eszközt
  2. Válassza ki a "HTML to PDF" opciót
  3. Töltse fel az összes HTML fájlt (index.html és az összes fejezet)
  4. Állítsa be a megfelelő beállításokat
  5. Generálja a PDF-et
Figyelem: Ez a módszer nem mindig működik megfelelően a helyi fájlokkal. Először próbálja meg a böngésző alapú megoldásokat.

4. Böngésző alapú megoldás - Egyszerű Könnyű

Firefox böngésző használatával

A Firefox böngésző képes automatikusan követni a linkeket és összefűzni az összes oldalt.

  1. Nyissa meg a Firefox böngészőt
  2. Nyissa meg a kézikönyv főoldalát: manual/index.html
  3. Nyomja meg a Ctrl+P (Cmd+P Mac-en)
  4. A nyomtatási beállításokban:
    • Válassza ki a "PDF mentése" opciót
    • Állítsa be a "Több oldal" opciót "Minden oldal" értékre
    • Kapcsolja be a "Háttér nyomtatása" opciót
    • Válassza ki az A4 oldalméretet
  5. Kattintson a "Mentés" gombra
Megjegyzés: Ez a módszer automatikusan követi a kézikönyv linkjeit és összefűzi az összes oldalt egy PDF-be.

5. Automatizált script a könnyebb használathoz

Bash script wkhtmltopdf-hez

Létrehozhat egy egyszerű scriptet, amely automatikusan generálja a teljes PDF-et.

#!/bin/bash # Speleo Studio kézikönyv PDF generálása echo "Speleo Studio kézikönyv PDF generálása..." # Ellenőrizze, hogy a wkhtmltopdf telepítve van-e if ! command -v wkhtmltopdf &> /dev/null; then echo "Hiba: wkhtmltopdf nincs telepítve!" echo "Töltse le innen: https://wkhtmltopdf.org/downloads.html" exit 1 fi # Navigáljon a manual mappába cd manual # Generálja a PDF-et wkhtmltopdf \ --enable-local-file-access \ --page-size A4 \ --margin-top 20mm \ --margin-bottom 20mm \ --margin-left 15mm \ --margin-right 15mm \ --header-html header.html \ --footer-html footer.html \ --toc \ --toc-header-text "Tartalomjegyzék" \ index.html \ 01-bevezetes.html \ 02-projekt-kezeles.html \ 03-adatok-importalasa.html \ 04-3d-vizualizacio.html \ 05-felmereek-szerkesztese.html \ 06-attributumok.html \ 07-eszkozok.html \ 08-exportalas.html \ 09-beallitasok.html \ 10-tippek-trukkok.html \ ../speleo-studio-teljes-kezikonyv.pdf echo "PDF sikeresen létrehozva: speleo-studio-teljes-kezikonyv.pdf"
  1. Mentse el a scriptet generate-pdf.sh néven
  2. Tegye futtathatóvá: chmod +x generate-pdf.sh
  3. Futtassa: ./generate-pdf.sh

Legjobb megoldás a legtöbb felhasználónak

wkhtmltopdf a legjobb megoldás egyetlen PDF fájl létrehozásához, mert:

  • Automatikusan összefűzi az összes oldalt

7. Hibaelhárítás

Gyakori problémák és megoldások

Probléma: A képek nem jelennek meg

Megoldás: Használja a --enable-local-file-access kapcsolót wkhtmltopdf-nél.

Probléma: A CSS formázás nem megfelelő

Megoldás: Próbálja meg a Firefox böngészőt, vagy használja a --print-media-type kapcsolót.

Probléma: Az oldaltörések nem megfelelőek

Megoldás: Állítsa be a megfelelő margókat és oldalméretet.

Probléma: A linkek nem működnek

Megoldás: Ellenőrizze, hogy minden fájl ugyanabban a mappában van-e.
